Program ini merupakan aplikasi desktop berbasis Java (Swing) yang digunakan untuk mengelola sistem booking lapangan futsal (Football Arena). Aplikasi ini dirancang untuk memudahkan pengguna dalam melakukan pemesanan lapangan, melihat daftar booking, serta mengelola data booking berdasarkan peran pengguna (Admin dan Customer).

Aplikasi menyediakan fitur login berbasis role, di mana Admin memiliki hak akses lebih luas dibandingkan Customer. Admin dapat menambah, mengedit, menghapus, serta melihat laporan booking, sedangkan Customer hanya dapat melakukan pemesanan dan melihat daftar booking yang tersedia.

Data booking disimpan secara persisten menggunakan file teks, sehingga data tetap tersimpan meskipun aplikasi ditutup. Program ini dibuat dengan konsep Object-Oriented Programming (OOP) dan memisahkan antara logika data, penyimpanan, dan tampilan antarmuka.
